# 学长笔记面板 · 设计

日期：2026-07-08 ｜ 状态：已确认（黑客松原型）

## 目标
右侧保留现有 RAG 聊天；左侧自动沉淀每轮问答为「学长笔记」卡片，可一键导出 Markdown FAQ。

## 决策
- 布局：`st.columns([2, 3])` 左笔记右聊天，全局 `layout="wide"`；`st.chat_input` 保持页面底部固定。
- 笔记生成：每轮回答结束后追加一次**非流式**小 LLM 调用，压缩成 2-3 条要点（每条 ≤30 字）；调用失败降级为截取答案前两句，不影响主流程。
- 数据：`st.session_state.notes = [{q, points, sources, time}]`，仅存内存（MVP 不落盘）。
- 导出：`st.download_button` 现场拼 Markdown（标题/日期/每条 Q+要点+来源链接），文件名 `学长组FAQ_<日期>.md`。
- 配色：米白底 `#FAF6EF`，青色主色 `#12857E`，暖橙点缀 `#E8833A`；笔记为白色圆角卡片 + 青色左竖条；导出按钮橙色。
- 改动范围：仅 app.py。

## 演示闭环
提问 → 右侧回答附来源 → 左侧自动记笔记 → 导出 FAQ，一键沉淀知识。
